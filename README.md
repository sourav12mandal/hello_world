# GitHub Hello World

A more complete example project to demonstrate best practices in open source.

## Features
- **CLI Support**: Pass a name via command-line arguments.
- **Unit Testing**: Automated testing with Python's built-in `unittest`.
- **CI/CD**: Automatic testing on GitHub Actions for every push.

## How to run
```bash
python main.py --name "Your Name"
```

## How to test
```bash
python -m unittest test_main.py
```
